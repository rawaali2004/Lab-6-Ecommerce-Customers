# Lab-6-Ecommerce-Customers
1. Introduction

This project aims to predict the Yearly Amount Spent by customers using machine learning techniques.

2. Data Exploration

The dataset was explored using head(), info(), and describe().
It contains numerical features such as session length, time on app, time on website, and membership length.

3. Data Cleaning

Missing values were checked using isnull() and handled using dropna().
The dataset was already clean.

4. Model Used

A Linear Regression model was used to predict customer spending.

5. Results
 • MAE: 7.4069
 • MSE: 83.3321
 • RMSE: 9.1286
 • R² Score: 0.9869

The model achieved 98.69% accuracy, indicating excellent performance.

6. Conclusion

The model performed very well and can accurately predict customer spending. 
Further improvements can be made using advanced models.
